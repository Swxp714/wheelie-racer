01 게임 컨셉

양손으로 바이크를 조종하고, 윌리로 속도를 높이되, 너무 높이 들면 뒤로 꼬꾸라져 사망하는 모션컨트롤 레이싱 게임

핵심 재미 요소

윌리 리스크/리워드
손을 높이 들수록 속도 UP, 과하면 사망 → 긴장감
직관적 조작
실제 핸들 잡듯 양손으로 조종 → 몰입감
도발 시스템
뻐큐 제스처로 도발 → 소셜 콘텐츠
복셀 아트
Smashy Road 스타일 → 가벼운 비주얼

게임 루프

FLOW시작 → 레이싱 → 윌리로 가속 → 장애물 회피
  ├── 균형 유지 성공 → 빠른 속도 → 높은 점수
  └── 균형 실패 → 뒤로 넘어짐 → 사망 → 리스폰 → 반복

뻐큐 감지 코드

PYTHONdef is_middle_finger(landmarks):
    """중지만 펴지고 나머지는 접힌 상태 감지"""
    middle_up  = landmarks[12].y < landmarks[10].y  # 중지 TIP < PIP
    index_down = landmarks[8].y  > landmarks[6].y   # 검지 접힘
    ring_down  = landmarks[16].y > landmarks[14].y  # 약지 접힘
    pinky_down = landmarks[20].y > landmarks[18].y  # 새끼 접힘
    return middle_up and index_down and ring_down and pinky_down

패턴 1: 손가락 접힘 감지

PYTHONFINGER_TIPS = [8, 12, 16, 20]
FINGER_PIPS = [6, 10, 14, 18]

def get_finger_states(landmarks):
    states = []
    for tip, pip in zip(FINGER_TIPS, FINGER_PIPS):
        states.append(landmarks[tip].y < landmarks[pip].y)
    thumb = landmarks[4].x < landmarks[3].x
    return [thumb] + states

패턴 2: 스티어링 각도

PYTHONimport math

def get_hand_tilt(landmarks):
    wrist = landmarks[0]
    mcp = landmarks[9]
    angle = math.atan2(mcp.y - wrist.y, mcp.x - wrist.x)
    return math.degrees(angle)
