## 현재 완성된 기능 (MVP)

| 카테고리 | 기능 |
|----------|------|
| 코어 | 1인칭 핸들바 뷰, 윌리 시스템, 스티어링 |
| 스턴트 | 백플립/리버스/더블/트리플, 에어타임 점수, 배수 시스템 |
| 제스처 | 뻐큐(홀드+뒤보기), 담배(피기+던지기), V사인 |
| 카메라 | 1인칭/3인칭 스턴트/사망 래그돌/뻐큐 뒤보기 자동전환 |
| 트랙 | 무한 절차적 생성, 장애물(콘/차), 점프대 3종(S/M/L), 코인 |
| AI | AI 레이서 3대 |
| 멀티 | PeerJS P2P, 방 생성/참가, TURN 릴레이 |
| HUD | 점수, 속도, 윌리 게이지, 스턴트 텍스트(크기/색/애니메이션) |
| 물리 | 래그돌 사망(충돌/윌리 분리), 바닥 바운스, 속도 비례 |
| 입력 | MediaPipe 손 인식 + 키보드 폴백 |

## Phase 2: 속도감 & 비주얼 이펙트

### 2.1 속도감 강화

**스피드 라인 (바람 효과)**
```
속도 20+ 부터 화면 바깥쪽에서 안쪽으로 흰색/회색 픽셀 라인
- CSS overlay로 구현
- 속도에 비례: 라인 개수, 길이, 속도 증가
- 화면 좌우 가장자리에서 시작, 중앙으로 수렴
- 속도 30+: 라인 더 많아지고 빨라짐
- 속도 40(최고속): 화면 약간 FOV 확대 + 색수차 효과
```

**카메라 FOV 변화**
```
정지: FOV 75
저속(~15): FOV 75
중속(15~30): FOV 80
고속(30~40): FOV 85 + 약간의 모션 블러 느낌
```

**엔진 사운드 (Web Audio API)**
```
- 기본 엔진 루프 (주파수 변조로 속도 표현)
- 윌리 시 엔진음 pitch 상승
- 스턴트 착지 시 타이어 스퀼 사운드
- 충돌 시 크래시 사운드
- 코인 수집 효과음
- 뻐큐 에어혼
```

### 2.2 비주얼 이펙트

**파티클 시스템**
```
- 뒷바퀴 먼지: 가속 시 뒷바퀴에서 갈색 파티클
- 윌리 연기: 윌리 시 뒷바퀴 흰 연기
- 부스트 불꽃: 최고속 시 배기관에서 불꽃
- 코인 수집: 금색 파티클 터짐
- 착지 충격파: 높이서 착지 시 원형 웨이브
```

**화면 이펙트**
```
- 비네팅: 고속 시 화면 가장자리 어두워짐
- 색수차: 최고속 시 RGB 분리 효과
- 화면 틸트: 스티어링에 따라 살짝 기울어짐
```

## Phase 3: 산악 맵 (절차적 생성 터레인)

### 3.1 터레인 시스템

**노이즈 기반 높이맵**
```
기술 스택:
- simplex-noise.js (CDN)
- PlaneGeometry 버텍스 디스플레이스먼트
- 높이 기반 버텍스 컬러링

생성 파이프라인:
1. Simplex noise로 높이맵 생성 (여러 옥타브 레이어)
2. 도로 경로를 CatmullRomCurve3 스플라인으로 정의
3. 도로 경로 주변 터레인 높이를 flatten
4. 터레인 메시 생성 (청크 단위)
5. 도로 메시를 터레인 위에 배치 (레이캐스트)
6. 소나무/바위 등 오브젝트 스캐터
```

**높이 기반 컬러링**
```
0~30%:  초록 (#4CAF50) - 풀밭
30~60%: 갈색 (#8B6914) - 흙/바위
60~80%: 회색 (#888888) - 암석
80~100%: 흰색 (#EEEEEE) - 눈
```

### 3.2 5가지 맵

| 맵 이름 | 테마 | 특징 | 난이도 |
|---------|------|------|--------|
| **GREEN VALLEY** | 초원 산길 | 완만한 경사, 넓은 도로, 초보용 | ★☆☆☆☆ |
| **PINE RIDGE** | 소나무 숲길 | 중간 경사, 좁은 커브, 나무 장애물 | ★★☆☆☆ |
| **ROCKY PASS** | 바위 협곡 | 급경사, 좁은 도로, 절벽 | ★★★☆☆ |
| **SNOW PEAK** | 설산 정상 | 미끄러운 도로, 폭풍 이펙트, 시야 제한 | ★★★★☆ |
| **DEATH MOUNTAIN** | 화산 지대 | 극한 경사, 용암 장애물, 무너지는 길 | ★★★★★ |

### 3.3 업힐 / 다운힐 모드

**업힐 모드**
```
- 산 아래에서 출발 → 정상까지 레이싱
- 속도가 느려짐 (오르막 중력 패널티)
- 윌리 유지가 더 어려움 (중력이 뒤로 당김)
- 스태미나 관리 중요
```

**다운힐 모드**
```
- 산 정상에서 출발 → 아래까지 레이싱
- 자연 가속 (내리막 중력 보너스)
- 브레이크 컨트롤 중요 (너무 빠르면 커브에서 이탈)
- 스턴트 기회 많음 (자연 점프 지점)
```
